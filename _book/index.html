箭頭函式arrow function
(1) 沒用花括號({})時，代表會自動加return，也只能在一行的語句的時候使用
(2) 使用花括號({})則是可以加入多行的語句，不過return不會自動加

//ex.1
function funA(){
  console.log('hello world!');
}
funA();
//es6後▼▼▼
const funA = () => {
  console.log('hello world!'); 
}
funA();

運算子
(1) && = 邏輯比較
(2) & = 2進制位元運算(乘法概念)
(3) || = 邏輯比較
(4) | = 2進制位元運算(加法概念)

console.log(1 && -2) //-2
console.log(1 && 0) //0
console.log(3 & 2) //2 11*10=10=2(上下乘)
console.log(1 | 2) //3 01+10=11=3
console.log(0 || 2) //2
